Catégorisation officielle · OpenTelemetry

La télémétrie OpenTelemetry de Claude Code apporte des dimensions impossibles à déduire des transcripts : main / sous-agent / auxiliaire, skill active et type d'agent. Pour l'activer, ajoute ce bloc dans ~/.claude/settings.json puis redémarre tes sessions Claude Code : 
"env": {
  "CLAUDE_CODE_ENABLE_TELEMETRY": "1",
  "OTEL_METRICS_EXPORTER": "otlp",
  "OTEL_EXPORTER_OTLP_PROTOCOL": "http/json",
  "OTEL_EXPORTER_OTLP_ENDPOINT": "http://localhost:3456",
  "OTEL_METRIC_EXPORT_INTERVAL": "30000"
}
Les métriques arrivent directement ici (récepteur OTLP intégré) — rien d'autre à installer. Si ce serveur est éteint, Claude Code ignore simplement l'export.
